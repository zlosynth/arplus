\section{Installation}

Arplus is 14 HP wide. It is powered by a +12V/{\textminus}12V 2\texttimes5
connector. The red stripe (\textminus12V) has to be connected on the side of the
board marked with the white line. The module must be mounted in a eurorack case.

\vspace{1cm}

\section{Quick start}

Turn all the knobs to the initial position illustrated on the front page of this
manual.
Click the \symbolArp{} (ARP), GROUP, and SCALE buttons until each of them lights
up the top-most LED.
Connect a clock source to the TRIGGER input, and connect the ROOT output to your
mixer.

\section{Arpeggio}

selected type defines how should be the notes ordered. Some types can be RESET
to their initial position. Other types are repeating the same sequence until
NEXT is triggered.

\section{Calibration}

Calibrate the TONE input.

\begin{packed_enumerate}
  \item While holding the RESET button, connect a jack to the TONE CV input.
  \item Play note C on the CV source and press the RESET button.
  \item Play C one octave higher and press the RESET button again.
  \item Successful calibration is confirmed by all LEDs litting up.
\end{packed_enumerate}

Calibrate the QUANT. ouput.

\begin{packed_enumerate}
  \item (optional) Complete TONE input calibration.
  \item While holding the \symbolArp (ARP) button, connect a cable between the
    QUANT. output and TONE input, then let go of the button.
\end{packed_enumerate}

\section{CV re-mapping}

Change the CV mapping.

\begin{packed_enumerate}
  \item Hold GROUP and SCALE buttons for 3 seconds.
  \item Turn potentiometer A to assign CV input C to attribute B.
  \item Click GROUP or SCALE button to exit the menu.
\end{packed_enumerate}

\section{Reset}

Calibration, CV mapping and all other attributes are persisted between restarts
of the module. To reset their values, hold the TONIC button while powering
the module on.
